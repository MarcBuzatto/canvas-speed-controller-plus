# Canvas Speed Controller Plus - Landing Page

Landing page oficial da extensão **Canvas Speed Controller Plus** para Chrome.

## 🚀 Deploy na Vercel

### Opção 1 - Deploy direto (recomendado)

1. Acesse [vercel.com/new](https://vercel.com/new)
2. Importe este repositório
3. Deixe todas as configurações padrão (Vercel detecta automaticamente)
4. Clique em Deploy

### Opção 2 - Via CLI

```bash
npm i -g vercel
vercel
```
